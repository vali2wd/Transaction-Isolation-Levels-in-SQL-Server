Project showcases the different levels of Isolation Levels available in SQL Server.

The database used in the demo is a 2010 Snapshot of Stack Overflow.<br/>
[Stack Overflow Snapshots through the Years](https://www.brentozar.com/archive/2015/10/how-to-download-the-stack-overflow-database-via-bittorrent/)<br/>
It is necessary to import such a file into your SQL Server instance to run the demo.<br/>
